มาเริ่มสร้าง API กันก่อนครับ
API ถ้าเอาง่าย ๆ แบบบ้าน ๆ แต่ไม่ถูกหลักการ ให้มอง เห็นภาพ เปรียบกับ PHP มันก็คือ ส่วน ของการติดต่อ ฐานข้อมูล ปรกติ ที่เราติดต่อกับ ฐานข้อมูล เวลาเราจะเรียกใช้ ข้อมูลที่เรา Connect ฐานมาได้ ก็จะเป็น $x = mysql_fetch_array($query) .... อะไร ก็ว่าไป มันก็จะได้ คืนมาเป็น $x[0] เป็น array ช่องของข้อมูล ถูกไหมครับ อันนี้ก็คล้าย ๆ กัน เพียงแต่ว่า มันจะคืน ข้อมูล ออกมา ในรูปแบบของ JSON Object // พยายามจะอธิบายให้เห็นภาพ จะได้ ไม่งง แต่ดูท่าทาง มันจะงง เอาไว้ ไปทำความเข้าใจ ในตอนพาทำเลยก็แล้วกัน // เอาแบบทางการ ก็ไปอ่านเอาที่นี่ แล้วกันครับ เขียนไว้ค่อนข้างดี Resful API คืออะไร
มาเริ่มติดตั้ง Express Resful API กันก่อน
ผมจะพาไปทางลัด เพื่อให้ง่ายในการติดตั้ง ผมจะพาใช้ generator-express-ts เพื่อให้ มันพา generate express api แบบ typescript ขึ้นมานะครับ (อธิบายตามความเข้าใจผมเด้อ ผิดถูก ขออภัย บอกแล้ว จะพาดำน้ำ)
